Introduction

Welcome to Issue Beacon. We respect your privacy and are committed to protecting your personal data. This privacy policy explains how we handle your information when you use our GitHub analytics platform.

Information We Collect

GitHub API Data

Issue Beacon fetches publicly available data from GitHub's API, including repository information, issues, commits, and contributor statistics. This data is only accessed when you explicitly request analysis of a specific repository.

Local Storage

We may store your search preferences and recently viewed repositories in your browser's local storage to enhance your experience. This data never leaves your device.

Analytics Data

We do not use any third-party analytics services. All data visualization and processing happens entirely in your browser.

How We Use Your Information

  • • To display GitHub repository analytics and visualizations
  • • To provide commit tracking and contributor insights
  • • To enhance user experience through local storage preferences

Data Security

Issue Beacon is a client-side application. All API requests are made directly from your browser to GitHub's servers. We do not store, transmit, or process your data on our servers because we don't have any.

Third-Party Services

Issue Beacon integrates with the following third-party service:

GitHub API

We use GitHub's public API to fetch repository data. Please refer to GitHub's Privacy Statement for more information about how GitHub handles data.

Your Rights

Since we don't collect or store personal data on our servers, you have complete control over your information. You can clear your browser's local storage at any time to remove any saved preferences.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated revision date.

Contact Us

If you have any questions about this privacy policy, please contact us through our GitHub repository .
