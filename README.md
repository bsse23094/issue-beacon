- **Without authentication:** 60 requests/hour
- **With Personal Access Token:** 5,000 requests/hour

### Adding Your GitHub Token

To avoid rate limiting issues, add a Personal Access Token:

1. **Generate a token:**
   - Visit [GitHub Settings â†’ Personal Access Tokens](https://github.com/settings/tokens)
   - Click "Generate new token (classic)"
   - Name it (e.g., "IssueBeacon")
   - **Don't select any scopes** (public repos only)
   - Click "Generate token" and copy it

2. **Add to environment:**
   ```typescript
   // src/environments/environment.ts
   export const environment = {
     production: false,
     githubApiBase: 'https://api.github.com',
     githubToken: 'ghp_your_token_here' // Paste your token here
   };
   ```
